Proyecto

El proyecto consiste en crear una aplicación web estilo blog programada en Python en Django. Esta web contendrá admin, perfiles, registro, páginas y formularios. Para hacer el blog versátil y con posibilidad de tener la mayor cantidad de opciones posibles, se decidió hacer el blog sobre el deporte más popular, el Fútbol.
